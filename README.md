## Slot Machine
Semplice Slot Machine in Java con interfaccia grafica.
Ogni volta che il tasto "GIOCA" viene premuto vengono generati 3 numeri casuali, il giocatore parte con 100 monete, ogni giocata costa 1 moneta, se 2 su 3 numeri sono uguali il giocatore guadagna 3 monete, se tutti e 3 i numeri sono uguali il giocatore fa jackpot e il gioco finisce.
